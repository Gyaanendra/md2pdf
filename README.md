# md2pdf

Markdown → PDF converter with live preview, syntax highlighting, and KaTeX math rendering.

## Features

- Split-pane live preview
- Syntax-highlighted code blocks (PrismJS)
- KaTeX math rendering
- One-click PDF export with header/footer
- Dark/light code themes

## Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/md2pdf)

## Local Dev

```bash
npm install
npm run dev
```

Open `http://localhost:3000`
